# Mind Intelligence Systems

**The umbrella architecture for agentic systems that help humans understand, structure, improve, and extend learning, self-understanding, psychology, neuroscience, research, and human development.**

Mind Intelligence Systems is the canonical portfolio for building, deploying, and evolving personal second brains and research intelligence tools. It follows a strict naming doctrine: **OS is lived. System is engineered. Systems is a category.**

## Purpose
This repo defines the entire swarm:
- Personal lived experience via **Agentic Mind OS**
- Reusable research portfolio via **Research Intelligence Systems**
- Domain-specific excellence in **Psychology** and **Neuroscience** Research Intelligence Systems
- Shared runtime via **Research Intelligence OS**
- Discovery layer via **Awesome Mind Agent Skills**
- Premium distribution via **Starlight Mind OS Pro**
- Grounding models in `models/human-mind/`

It ensures consistency, interoperability, and excellence across the ecosystem using latest best tech: structured JSON schemas, ReAct-style agents, human-mind modular models, preregistration/Registered Reports workflows, BIDS/NWB/MNE standards, and agent-readable manifests (repo-mesh.yaml, researchpack.yaml, etc.).

## How People Experience It
Humans (students, creators, researchers, founders) experience it as a coherent ecosystem:
- Start in Agentic Mind OS for daily reflection, canon building, and personal growth.
- Transition seamlessly into research packs for literature synthesis, evidence tables, and claim graphs.
- Use domain systems for deep work in psychology (construct mapping, psychometrics) or neuroscience (BIDS pipelines, NWB datasets).
- Leverage Research OS for custom agents and reproducible workflows.
- Premium users get guided onboarding via Starlight Pro.

**Daily UX**: Open vault → capture → distill with agents → review → contribute to research pack. Onboarding is 7-day guided with templates.

**Key Flows**: Personal canon → evidence synthesis → reproducible pipeline → published insight. All grounded in human mind models (attention, memory, belief, metacognition, etc.).
