# Behavior

## Definition
Observable actions and patterns of action that result from the interaction of internal states and external contexts.

## Key Processes
- Habit formation and automaticity
- Goal-directed vs. habitual control
- Behavioral activation / inhibition
- Context and cue control
- Feedback loops (behavior → outcome → learning)

## Related Constructs
Motivation, emotion, learning, decision-making, identity (identity-behavior gap).

## Research Notes
Habit loop (cue-routine-reward), dual-process theories, behavioral economics (nudges), reinforcement learning accounts.

## Agent Prompts / Usage
- Daily mind capture: log actions and contexts.
- Pattern analyst: detect habit loops and friction points.
- Weekly review: measure behavior change against goals.
- Skills for behavior design (environment shaping, implementation intentions).
