# Memory

## Definition
Systems for encoding, storing, and retrieving information and experiences over time.

## Key Processes
- Encoding (shallow vs. deep)
- Storage (short-term, working, long-term)
- Retrieval (recall, recognition, reconstruction)
- Consolidation and reconsolidation
- Forgetting and interference

## Subsystems
- Working memory
- Episodic memory
- Semantic memory
- Procedural memory
- Prospective memory

## Related Constructs
- Learning, belief formation, identity (autobiographical memory), emotion (flashbulb, mood-congruent)

## Research Notes
Atkinson-Shiffrin, Baddeley & Hitch working memory, levels of processing, constructive memory (Loftus).

## Agent Prompts / Usage
- Distill sources into atomic notes for better encoding.
- Active recall generators target retrieval practice.
- Personal canon builder strengthens semantic/episodic links.
- Weekly review supports consolidation.
