# Emotion

## Definition
Affective states involving physiological arousal, cognitive appraisal, and behavioral tendencies.

## Key Processes
- Appraisal
- Expression and regulation
- Emotional contagion
- Mood vs. emotion vs. affect

## Related Constructs
- Motivation (approach/avoid), attention (bias), memory (enhancement), decision-making (somatic markers), behavior.

## Research Notes
Basic emotions (Ekman), dimensional (valence/arousal), appraisal theories (Lazarus, Scherer), constructed emotion (Barrett).

## Agent Prompts / Usage
- In daily capture: tag emotional valence and intensity.
- Pattern analyst: identify emotional patterns linked to productivity or learning blocks.
- Memory governor: note emotion during encoding for better retrieval cues.
- Guard: never interpret as clinical; stay descriptive.
