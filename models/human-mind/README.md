# Human Mind Model

Modular ontology for agentic systems that model human cognition, affect, and behavior.

**Status**: v0 foundation. Modular Markdown specs. Designed for agent consumption and schema derivation.

**Guardrails**:
- This is a descriptive/research model, not a clinical or diagnostic tool.
- Separate observation, interpretation, hypothesis, evidence.
- No individual diagnosis language.

Reference these files when:
