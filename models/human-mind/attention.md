# Attention

## Definition
The selective allocation of cognitive resources to specific stimuli, thoughts, or actions while filtering others.

## Key Processes
- Selective attention
- Sustained attention / vigilance
- Divided attention
- Executive attention / attentional control
- Orienting and alerting

## Related Constructs
- Working memory capacity
- Cognitive load
- Mindfulness / meta-attention
- Attentional bias (in emotion and belief)

## Research Notes
Classic models: Broadbent filter, Treisman attenuation, Posner cueing, Baddeley central executive.

Modern: predictive processing accounts, resource models vs. selection models.

## Agent Prompts / Usage
When mapping user notes or sources:
- Identify what the user is attending to vs. ignoring.
- Detect attentional bottlenecks in learning or review workflows.
- Suggest attention-management skills (e.g., single-tasking, environment design).
