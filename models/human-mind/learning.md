# Learning

## Definition
Relatively permanent change in knowledge, skills, or behavior resulting from experience.

## Key Processes
- Encoding and elaboration
- Retrieval practice / testing effect
- Spaced repetition
- Interleaving and varied practice
- Transfer and generalization
- Metacognitive monitoring of learning

## Related Constructs
Memory, attention, motivation, metacognition, belief (epistemic).

## Research Notes
Ebbinghaus forgetting curve, spacing effect, testing effect (Roediger), desirable difficulties, 4-stage model of competence.

## Agent Prompts / Usage
- Active recall generator skill.
- Source distillation into atomic, linked notes.
- Learning architect agent designs personalized spaced systems.
- Weekly review measures learning velocity and retention.
