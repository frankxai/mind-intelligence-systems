# Contributing to Mind Intelligence Systems

Thank you for contributing to the Mind Intelligence Systems swarm.

## Code of Conduct

- Respect the naming doctrine at all times.
- Keep research claims source-grounded.
- No clinical or diagnostic language.
- Separate observation from interpretation.

## How to Contribute

1. Check existing issues and `.codex/tasks.md`.
2. For foundation work, branch from `main` to `hermes/foundation-vX`.
3. Follow the structure in AGENTS.md and HERMES.md.
4. Update cross-repo links and mesh when relevant.
5. Open PR with clear description of what was added and why.

## Pull Request Process

- Title: `Foundation v0: ...` or conventional.
- Body must reference which repo(s) and Codex tasks.
- Small, focused changes preferred.
- All new model files must follow the template in models/human-mind/.

## Codex / Agent Contributions

Agents are encouraged. Every PR should leave the repo more agent-readable.

See `.codex/tasks.md` for the current queue.

## Questions

Open an issue with label `question` or discuss in relevant repo.
