# Architecture

Mind Intelligence Systems is a layered portfolio:

1. **Personal Layer** — Agentic Mind OS (lived daily practice, vault, agents, skills)
2. **Domain Research Layers** — Psychology RIS, Neuroscience RIS (specialized agents, skills, schemas, workflows)
3. **Reusable Runtime Layer** — Research Intelligence OS (contracts, templates, evals, pipeline grammar)
4. **Portfolio Layer** — Research Intelligence Systems (cross-domain packs, workflows, claim graphs)
5. **Discovery Layer** — Awesome Mind Agent Skills (curated ecosystem)
6. **Premium Distribution** — Starlight Mind OS Pro (onboarding, dashboards, workshops)
7. **Canon & Governance** — This repo (naming, models, mesh, doctrine)

## Key Contracts

- MindPack: packaging for agents, skills, workflows, schemas, vaults.
- ResearchPack: for research systems.
- Agent contracts and pipeline contracts defined in Research Intelligence OS.

## Data & Privacy

- Local-first vaults for personal data.
- Source-grounded research artifacts.
- Schemas for interoperability (JSON Schema where applicable).

## Agent Interaction Model

All repos are designed for:
- Codex / Claude Code / Gemini / Hermes agents
- `.codex/tasks.md` queues
- AGENTS.md and HERMES.md for context
- Issue templates for task intake

See individual repo ARCHITECTURE or README for domain specifics.
